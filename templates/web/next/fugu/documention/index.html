Requirements

There are system requirements in order to install and setup this template and its components properly. Make sure that you are running the node.js & npm install

Installing Development Environment

We require many packages (dependencies) to run our site. Move into /xmooze (project root) directory and run command below. 

yarn

After the installation, you should have a folder named /node_modules in the project root directory.

How To Run Development Server

To start our development server run command below:

yarn dev

Open your browser and visit http://localhost:3000. You should see a page like below.

CLI Commands

  1. yarn dev to Start development server at http://localhost:3000
  2. yarn build for Generating production build
  3. yarn satrt To serve build files at http://localhost:3000
  4. yarn export for Generating static site
