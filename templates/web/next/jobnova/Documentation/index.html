Jobnova - Next.js Job Board & Job Portal Template

Introduction

Jobnova is a modern, clean and intuitive React js template. Jobnova is designed for job board websites. The template is based on Latest Next 14, React 18 & Bootstrap 5 and includes many unique pages and components that look perfect on any device. Jobnova is represents simple design to help build the website for advertising vacancies, finding suitable staff, receiving employer’s resumes and CV. it will perfectly suit for projects of direction: Job Search, Directory Portal, Resume & CV, Recruitment Agency and the like. Jobnova has super clean simple, easy customizable react components and widgets etc… This job posting template comes with a awesome unique latest new design also we ensure you can easily design any type of job classified website template for jober and recruiter.

Command Description

How to change the fonts family in Jobnova?

Please follow the step by step process

Step 1

Open the layout.js (path: Jobnova_Nextjs/src/app/layout.js) file and please import the font on top of the page import { Plus_Jakarta_Sans } from 'next/font/google'

Step 2

Import your font family in layout.js (path: Jobnova_Nextjs/src/app/layout.js) file, or replace your font name in layout.js (path: Jobnova_Nextjs/src/app/layout.js) file.

import { Plus_Jakarta_Sans } from 'next/font/google' - Themes primary font family is Plus_Jakarta_Sans
Step 3

Pass your font family in layout.js (path: Jobnova_Nextjs/src/app/layout.js) file, or replace your font name in layout.js (path: Jobnova_Nextjs/src/app/layout.js) file.

const jakarta = Plus_Jakarta_Sans({
  subsets: ['latin'],
  weight: ["200','300','400','500','600', '700','800"],
  display: "swap",
  variable: '--font-jakarta',
})

- Themes primary font family is Plus_Jakarta_Sans
Step 4

change your font family in _variables.scss(path: Jobnova_Nextjs/src/app/assets/scss/_variables.scss) file

$font-family-base:     var(--font-jakarta);
$font-family-secondary:     var(--font-jakarta);

How to change the Theme primary color in Jobnova?

If you change the theme or template color please follow the below step, Please first of all open the _variables.js (path: src/app/assets/scss/_variables.scss) find the $primary and replace or changes with color code

like:
$primary:     #3b82f6; replace only #3b82f6 to your color (#f89d36)
